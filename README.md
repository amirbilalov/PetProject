# Hotel Booking API

REST API для управления бронированиями отелей. Учебный проект на ASP.NET Core с чистой архитектурой.

## 🛠 Технологии

- **C# / .NET 8** (или .NET 6/7 — укажи свою версию)
- **ASP.NET Core Web API**
- **Entity Framework Core** (Code First)
- **PostgreSQL** (развернут в Docker)
- **Архитектура**: чистая / слоеная архитектура (уточни, какую именно использовал)

## 🚀 Функциональность

### Администратор
- ✅ Создание комнаты (класс, стоимость, описание)
- ✅ Удаление комнаты
- ✅ Просмотр всех комнат
- ✅ Просмотр конкретной комнаты

### Пользователь
- ✅ Просмотр доступных комнат
- ✅ Бронирование комнаты (с проверкой на существующие брони)
- ✅ Просмотр информации о своем бронировании
- ✅ Защита от двойного бронирования (нельзя забронировать уже занятую комнату)
